## 1.1. Main Screen
HAML has 5 main tabs. They are:
1. Library
2. Search
3. Add
4. Edit
5. Delete

## 1.2. Library
### 1.2.1. `initDatabase`
Create database on local machine.
```zsh
./initDatabase
```
### 1.2.2. `removeDatabase`
Remove database on local machine.
```zsh
./removeDatabase
```
